  启动说明

  推荐方式

  - 直接双击项目根目录里的：

    start-gongkao-manager.bat

  这个脚本会自动：

  - 检查依赖
  - 自动构建最新版本
  - 启动本地网页服务
  - 自动打开浏览器

  之后如果你更新了代码，不需要再手动执行 build 和 start，直接再次双击这个脚本即可。

  补充方式

  - 如果你只是想基于当前已有构建快速预览，可以双击：

    start-gongkao-manager-prod.bat

  手动命令

  - 正式启动：

    npm run build
    npm run start:local

  说明

  - 启动后那个黑色终端窗口不要关，关掉后网页服务会停止。
  - 不用时，在终端里按 Ctrl + C 结束服务。
