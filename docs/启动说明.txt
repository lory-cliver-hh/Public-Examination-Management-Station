  启动说明

  1. 打开终端，进入项目目录：

  cd /d D:\公考记录系统

  2. 启动网页服务：

  npm run start -- --hostname 127.0.0.1 --port 3001

  3. 浏览器打开：

  http://127.0.0.1:3001/

  补充

  - 如果启动时报错找不到依赖，先执行一次：

  npm install

  - 如果还是打不开，再执行一次：

  npm run build

  然后重新运行启动命令。

  - 启动后那个黑色终端窗口不要关，关掉网页就会停止。
  - 不用时，在终端里按 Ctrl + C 结束服务。
