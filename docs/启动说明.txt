  启动说明

  推荐方式

  - 直接双击项目根目录里的：

    start-gongkao-manager.bat

  这个脚本会自动：

  - 检查依赖
  - 启动本地开发服务
  - 自动打开浏览器

  之后如果你继续改代码，页面会自动热更新，不需要每次再手动执行启动命令。

  补充方式

  - 如果你只是想用“正式启动”方式预览，而不是开发模式，可以双击：

    start-gongkao-manager-prod.bat

  手动命令

  - 开发模式：

    npm run dev:local

  - 正式启动：

    npm run build
    npm run start:local

  说明

  - 启动后那个黑色终端窗口不要关，关掉后网页服务会停止。
  - 不用时，在终端里按 Ctrl + C 结束服务。
